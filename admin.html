Biaya Admin Deposit OTP (Nokos)
Biaya admin yang ditambahkan ke setiap deposit OTP. Pembeli melihat: Nominal deposit + biaya admin.
Contoh: user mau isi Rp10.000 → total yang dibayar = Rp10.000 + biaya admin. Saldo yang masuk ke akun mereka tetap Rp10.000.
Markup Harga Order OTP
Markup nominal (Rp) yang ditambahkan ke harga setiap order OTP.
Contoh: harga supplier Rp500 + markup Rp200 = user melihat & membayar Rp700. Selisih Rp200 adalah keuntungan kamu.
Perubahan berlaku real-time tanpa perlu restart server.
Isi 0 untuk tidak menambah biaya. Tidak bisa diatur per-persentase.
Markup Harga Kayz Market Prem
Markup nominal (Rp) yang ditambahkan ke harga setiap produk Kayz Market Prem.
Contoh: harga supplier Rp50.000 + markup Rp5.000 = user melihat & membayar Rp55.000. Selisih Rp5.000 adalah keuntungan kamu.
Produk Kayz Market Prem otomatis muncul di kategori PREMIUM di halaman toko.
Isi 0 untuk tidak menambah biaya. Berlaku untuk semua produk Kayz Market Prem.

API Bot Sewa (Private)klik untuk buka
API privat (butuh key) berisi data sewa bot — bukan notifikasi biasa. Setiap order sewa bot yang LUNAS otomatis tercatat lengkap dengan link grup, durasi, dan waktu kadaluarsa. Bot kamu memakai API ini untuk otomatis join grup saat ada sewa baru, lalu keluar saat masa sewa habis (ada timer-nya).
Key bersifat rahasia. Generate ulang akan langsung menonaktifkan key lama. Simpan key di konfigurasi bot kamu.
Cara pakai (untuk bot)
Sertakan key lewat header x-api-key: <KEY> atau query ?key=<KEY>.

1. Daftar sewa aktif (polling)
GET /api/sewabot/list → { ok, count, serverTime, data:[...] }
Tambah ?all=1 untuk ikut sertakan yang sudah kadaluarsa.

2. Realtime stream (SSE, tanpa delay)
GET /api/sewabot/stream?key=<KEY>
Event: snapshot (saat connect) → rental (sewa baru) → expired (masa habis). Ada ping & reconnect berkala.

Field tiap item
id, groupUrl, buyerName, days, totalBayar, startedAt, expiresAt, remainingMs, status (active/expired).

Logika bot: saat dapat item baru → join groupUrl. Saat remainingMs habis / event expired → keluar grup.
